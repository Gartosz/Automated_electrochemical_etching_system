Zautomatyzowany system elektrochemicznego trawienia ostrzy wolframowych dla potrzeb laboratorium EC-STM pozwala na kontrolę i nadzór procesu trawienia ostrzy wolframowych. Dostępne są tryby AC, DC i VCCS, ustawianie napięcia, natężenia oraz częstotliwości prądu czy długości wysuwanego drutu. Przebieg sygnału przedstawiony jest na wykresie, a obok niego widnieją zmierzone wartości. System umożliwia precyzyjne przygotowanie ostrych sond wolframowych wymaganych w elektrochemicznej mikroskopii tunelowej (EC-STM). Monitorowanie parametrów trawienia w czasie rzeczywistym zapewnia powtarzalną jakość ostrzy i pozwala na optymalizację procesu.

Automated electrochemical tungsten tip etching system for EC-STM laboratory use enables control and monitoring of the etching process. Available modes include AC, DC, and VCCS, with adjustable voltage, current intensity and frequency, as well as wire extension length. The signal waveform is displayed on a graph with measured values shown alongside. The system enables precise preparation of sharp tungsten tips required for electrochemical scanning tunneling microscopy (EC-STM) applications. Real-time monitoring of etching parameters ensures reproducible tip quality and allows for optimization of the process.
